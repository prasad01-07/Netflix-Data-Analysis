# Netflix Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix content data to discover trends in movies and TV shows.

## Dataset
- netflix_titles.csv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Content Type Analysis
- Rating Distribution Analysis
- Country-wise Content Analysis
- Year-wise Content Trends
- Data Visualization


## Key Insights
- TV-MA is the most common Netflix rating.
- Movies dominate the Netflix catalog.
- Netflix content increased significantly after 2015.

## Files
- Copy_of_Netflix_Data_Analysis.ipynb
- netflix_titles.csv
- README.md  

## Key Insights

1. TV-MA is the most common content rating on Netflix.
2. Movies significantly outnumber TV Shows in the Netflix catalog.
3. Netflix content additions increased rapidly after 2015.
4. The United States contributes the largest amount of content.
5. Drama and International content dominate the platform.

## Conclusion

This project explored Netflix's Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Seaborn. Through data cleaning, exploratory analysis, and visualization, meaningful insights were extracted regarding content trends, ratings, countries, and release patterns.
